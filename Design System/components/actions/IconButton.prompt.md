**IconButton** — round, icon-only control for quiet chrome (close a sheet, go back); requires an accessible `label`.

```jsx
<IconButton icon="x" label="Close" />
<IconButton icon="chevron-left" label="Back" variant="onDark" />
```

Variants: `quiet` (default well fill), `onDark` (over the breather gradient), `accent` (tinted). Never below 44px.
