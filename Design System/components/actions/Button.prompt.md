**Button** — the primary action control; sentence case, honest verbs, ≥44px, full-width by default.

```jsx
<Button variant="primary" onClick={start}>It's happening</Button>
<Button variant="ghost" fullWidth={false}>Not now — stay anonymous</Button>
```

Variants: `primary` (Steady green), `repair` (clay, for after-the-storm moments), `secondary` (tinted fill), `ghost` (quiet — use for real declines, never a shame link), `crisis` (muted red, rare). Sizes `lg`/`md`/`sm`; keep primaries `md`+. Press settles to scale 0.98.
