**Icon** — a Lucide glyph tinted with `currentColor`; use sparingly, always beside text, never alone.

```jsx
<span style={{ display: "inline-flex", alignItems: "center", gap: 8 }}>
  <Icon name="wind" size={20} />
  Take a breather
</span>
```

Common names: `wind` (breather), `heart-handshake` (repair), `shield` / `moon`, `chevron-right`, `x`, `check`, `life-buoy` (crisis). No emoji anywhere in the system — use this instead.
