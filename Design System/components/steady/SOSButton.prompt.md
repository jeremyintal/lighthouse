**SOSButton** — the hero. One big, soft, gently breathing button the parent taps mid-meltdown.

```jsx
<SOSButton label="It's happening" sublabel="Tap for a breather" onClick={startBreather} />
```

Fully round, breathes on an 8s cycle, carries the one colored shadow in the system. Keep it large (≥200px) and in the bottom half of the screen (one-handed law). Only ever one per screen.
